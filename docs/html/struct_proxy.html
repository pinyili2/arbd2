Detailed Description

template<typename T, typename Enable = void>
struct Proxy< T, Enable >

Default constructor initializes the location to a default CPU resource and the address to nullptr.

Overloaded operator-> returns the address of the underlying object.

Returns
The address of the underlying object.

The resource associated with the data represented by the proxy.

Template class representing a proxy for the underlying data.
